# Alibaba

阿里巴巴集团（Alibaba Group）是中国的跨国科技公司，开源了多个技术项目，包括 Arthas。

## 开源项目

### Arthas

[[concepts/arthas|Arthas]] 是阿里巴巴开源的 Java 应用诊断调试工具，支持在线排查问题，无需重启应用。

- 版本 4.1.8+ 原生支持 [[model-context-protocol-mcp|Model Context Protocol (MCP)]]
- 允许 AI 客户端直接连接诊断 Java 应用问题

#### 1. springboot接入atrhas

###### 1.1 pom.xml 配置
~~~ pom.xml
<dependency>  
    <groupId>com.taobao.arthas</groupId>  
    <artifactId>arthas-spring-boot-starter</artifactId>  
    <version>4.1.8</version>  
</dependency>
~~~


- 4.1.8支持MCP
- 
##### 1.2 配置服务

~~~ yml
arthas:  
  mcpEndpoint: /mcp  --配置mcp 
  httpPort: 8563  
  password: 123456
~~~


###### 1.3界面访问
url: http://localhost:8563/ui/#/dashboard

![[1.ui.png]]



### 2.接入MCP

###### 2.1 配置mcp.json
~~~ json

{
  "mcpServers": {
    "arthas-mcp": {
      "type": "streamableHttp",
      "url": "http://localhost:8563/mcp",
      "headers": {
        "Authorization": "Bearer 123456"
      }
    }
  }
}

~~~
